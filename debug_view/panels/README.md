# Debug Panels

Panel notes for the visual debugger:

- `raw`: camera frame.
- `binary`: thresholded black-frame mask.
- `edges`: Canny edge output.
- `combined`: morphology and edge merge.
- `contours`: candidate rectangles.
- `perspective`: rectified target view.
- `pose`: PnP axes and relative angle overlay.
- `serial`: exact packet sent to the gimbal.
