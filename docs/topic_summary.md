# Topic Summary

Task mapping:

- `basic_trace_only`: basic requirement 1, tracing only, aiming off.
- `basic_static_aim`: basic requirement 2, aim and hit center within 2 s.
- `basic_track_aim`: basic requirement 3, aim from specified track position within 4 s.
- `adv_one_lap`: advanced requirement 1, one lap with continuous illumination.
- `adv_two_laps`: advanced requirement 2, two laps with continuous illumination.
- `adv_circle_sync`: advanced requirement 3, synchronized 60 mm circle trajectory.
